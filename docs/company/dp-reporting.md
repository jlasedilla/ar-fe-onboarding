# Reporting

## Daily Progress

Email your daily progress end-of-day:

```
--------------------------------------------------
Subject: [Daily Report] <YYYY/MM/DD> - <Nickname>
To: <TBD>
--------------------------------------------------

Content:
--------------------------------------------------
Completed Task(s) for the day

    - Task 1 [100% completed]
    - Task 2 [50% completed]

Target Task(s) for the next working day

    - Continue Task 2
--------------------------------------------------
```
